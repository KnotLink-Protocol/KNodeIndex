# Everything 搜索节点

基于 [Everything](https://www.voidtools.com/) 的文件搜索节点，通过 KnotLink OpenSocket 协议提供文件搜索、结果计数和打开文件夹功能。

## 仓库地址

- **GitHub**: https://github.com/hxh230802/Everything_node

## 下载地址

- https://github.com/hxh230802/Everything_node/releases/latest

## 功能特性

- **文件搜索** — 使用 Everything 引擎搜索文件，返回路径列表
- **结果计数** — 仅获取搜索结果数量
- **打开文件夹** — 打开第一个搜索结果的所在文件夹

## 快速开始

### 环境要求

- Windows 10+
- [Everything](https://www.voidtools.com/) 已安装并运行

### 安装

```bash
# 下载最新 release 中的 Everything_node.exe
```
