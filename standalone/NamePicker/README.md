# RandomPicker

随机点名工具，通过 KnotLink OpenSocket 协议提供单人/多人随机抽取、学生名单管理及点名事件信号。

## 仓库地址

- **GitHub**: https://github.com/hxh230802/NamePicker

## 下载地址

- https://github.com/hxh230802/NamePicker/releases/latest

## 功能特性

- **随机点名** — 单人随机抽取、多人批量抽取
- **名单管理** — 查询名单、添加/删除/更新学生信息
- **点名事件信号** — 点名触发时主动推送学生信息
- **窗口控制** — 主窗口和托盘图标的显示/隐藏控制
