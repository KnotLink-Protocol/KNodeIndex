# ClassIsland

课程信息与通知服务节点，通过 KnotLink OpenSocket 协议提供当前/下节课程查询、桌面通知提醒及课程事件信号。

## 仓库地址

- **GitHub**: https://github.com/hxh230802/ClassIsland

## 下载地址

- https://github.com/hxh230802/ClassIsland/releases/latest

## 功能特性

- **课程查询** — 获取当前课程、下节课程信息（课程名、教师名）
- **桌面通知** — 推送遮罩和正文提醒，支持语音播报
- **课程事件信号** — 上课/下课/放学等事件主动推送
