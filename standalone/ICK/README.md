# ICK

提供屏幕批注相关功能（Ink Canvas 画布控制、函数绘制等）。

## 仓库地址

- **GitHub**: https://github.com/hxh230802/ICK

## 下载地址

- https://github.com/hxh230802/ICK/releases/latest

## 功能特性

- **墨迹画布控制** — 显示/隐藏画布、切换笔/橡皮擦/光标模式、清空/撤销/重做
- **屏幕批注** — 屏幕模式（透明背景）、白板/黑板模式
- **悬浮工具栏** — 移动到屏幕各方位、展开/收起
- **函数绘制** — 支持 LaTeX 表达式绘制数学函数图像
